Installation

01. FTP Upload

  • Open Your FTP manager and connect to your hosting
  • Browse to required directory(Normally public_html)
  • Upload the files inside Your_Folder_Name folder
  • That's all !

02. All HTML Pages

  • index.html for Home Page
  • about-us.html for about page
  • category-listing.html for collection page
  • faq.html for faq page
  • category-product.html for the collection details page
  • single-product.html for the product page
  • wishlist.html for wishlist Page
  • blog-listing.html for Blog Page
  • blog-details.html for Blog Details Page
  • contact-us.html for Contact Page
  • checkout.html for checkout Page
  • login.html for user Page
  • order-tracking.html for Error Page

03. Fonts

Go to Google Fonts and select your favorite fonts and then just replace the code in big-tech/index.html file

Components customization
 



                            
01. Section structure

                            
Here is your section structure.

                            

<section class="visor_virtual_section top_padding">
    <div class="container visor_virtual"> 
        <div class="row"> 
            <div class="col-lg-12 pb-4 pb-lg-0">
            ........ 
            </div>
        </div>
    </div>
</section>

    

                            
In the class="visor_virtual_section" you can replace with your desired class.

                            
top_padding  is used for the space top of the sections.

                            
bottom_padding  is used for the space bottom of the sections.

                            
And in the section-header div you can add your heading of sections and tagline of sections.

                            
You can add your content on the 2nd .section-wrapper div.
